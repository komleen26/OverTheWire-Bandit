## 📖 About OverTheWire

**OverTheWire Bandit** is a beginner-friendly Linux wargame designed to teach essential command-line skills and problem-solving techniques required in cybersecurity.

It introduces concepts such as:

- Linux file system navigation
- File permissions
- Hidden files
- Special filenames
- Searching for files
- SSH
- File identification
- Text processing
- Basic scripting
- And much more...

Official Website:
https://overthewire.org/wargames/bandit/
