#Bandit level 0 to level 1:

#Commands used:
ssh bandit0@bandit.labs.overthewire.org -p 2220

#Command breakdown:
- `ssh` – Starts a Secure Shell (SSH) session to connect to a remote machine securely.
- `bandit0` – The username for the current level.
- `@` – Separates the username from the remote host.
- `bandit.labs.overthewire.org` – The hostname of the Bandit server.
- `-p` – Specifies the port number to connect to.
- `2220` – The SSH port used by the Bandit server.
