#Command breakdown:
- `cat` – Displays the contents of a file.
- `--` – Indicates the end of command options. Anything after this is treated as a filename, even if it begins with a hyphen (`-`).
- `"--spaces in this filename--"` – The filename enclosed in double quotes because it contains spaces.
